Project Aria Tools provides Python and C++ APIs to access the [Aria Digital Twin (ADT) dataset](https://www.projectaria.com/datasets/adt/) ([paper](https://arxiv.org/abs/2306.06362)).
This section provides a detailed walk through of:

* [Getting started](/docs/open_datasets/aria_digital_twin_dataset/getting_started)
    * A quickstart to install Project Aria Tools python package and run Aria Digital Twin tutorial notebook to access and visualize ADT ground-truth data.
* [Dataset download](/docs/open_datasets/aria_digital_twin_dataset/dataset_download.mdx)
   * A walkthrough of using `adt_benchmark_dataset_downloader` to download the published ADT dataset.
* [Visualizers](/docs/open_datasets/aria_digital_twin_dataset/visualizers)
   * Compile and run our visualizer using an example that accesses ADT data in C++.
* [Advanced tutorial](/docs/open_datasets/aria_digital_twin_dataset/advanced_tutorials)
  * A guide to learn how device synchronization works in ADT and run through a Jupyter notebook.
* [Data loader](/docs/open_datasets/aria_digital_twin_dataset/data_loader)
   *  APIs to load ADT data with handy code snippets
* [Data format](/docs/open_datasets/aria_digital_twin_dataset/data_format)
    *  How ADT data is organized and stored
